# Bank Eruptcity Coin 🚀

Welcome to **Bank Eruptcity Coin (BEC)**, a meme coin on the Solana blockchain! Our goal is to build a financial ecosystem for those in need.

## 🔹 **Project Features**
- **Roadmap Display**: Tracks milestones for BEC growth.
- **Interactive Trading Bot**: AI-based trading assistance.
- **USDC Airdrop System**: Rewards for holders at $2M market cap.
- **Virtual Agent**: Future AI stock/crypto prediction model.
- **Falling Money Animation**: Engaging website effects.

## 🚀 **Getting Started**
### **1️⃣ Install Dependencies**
```bash
npm install
```

### **2️⃣ Run the Project**
```bash
npm start
```

### **3️⃣ Build for Deployment**
```bash
npm run build
```

## 🌐 **Deployment**
### **GitHub Pages**
1. Run `npm run build`
2. Push to GitHub repository
3. Enable GitHub Pages from the repository settings

### **Netlify Deployment**
- Uses **`netlify.toml`** configuration for easy deployment.
